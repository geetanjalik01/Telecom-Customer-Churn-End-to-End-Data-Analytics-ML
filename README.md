# Telecom-Customer-Churn-End-to-End-Data-Analytics-ML
End-to-End Churn Analysis project using SQL Server for ETL, Python (Random Forest) for predictive modeling, and Power BI for executive-level storytelling.
## Project Overview
This project addresses the critical business challenge of Customer Churn in the Telecom industry. By building a complete data pipeline—from raw SQL data to a predictive Machine Learning model and an interactive Power BI dashboard—this project identifies why customers leave and predicts which customers are at high risk of churning next.

# Project Architecture
The project follows a 4-stage professional workflow:
ETL & Database Management (SQL Server): Data cleaning and transformation using SQL.
Exploratory Data Analysis (Power BI): Visualizing historical churn trends and KPIs.
Predictive Modeling (Python): Building a Random Forest Classifier to predict future churn.
Actionable Insights: Identifying high-risk segments to drive retention strategies.

#Tech Stack & Tools
Database: Microsoft SQL Server (T-SQL)
Data Analysis & ML: Python (Pandas, Scikit-Learn, NumPy, Jupyter Notebook)
Visualization: Power BI Desktop (DAX, Power Query)
Model: Random Forest Classifier (Accuracy: 85%)
